# 生产设备报修、检修、物资管理平台 启动说明

## 后端服务

### 启动命令
```bash
# 进入后端目录
cd backend

# 安装依赖（首次运行）
npm install

# 启动服务
npm start
```

## 前端服务

### 启动命令
```bash
# 进入前端目录
cd frontend

# 安装依赖（首次运行）
npm install --legacy-peer-deps

# 启动开发服务器
npm run serve
```

### 访问地址
- 前端应用：http://localhost:8080
- 登录页面：http://localhost:8080/login

## 常用操作

### 停止服务
```bash
# Windows系统：终止所有Node.js进程
taskkill /F /IM node.exe

# 或使用端口号终止进程
# 后端端口：3001
netstat -ano | findstr :3001
# 找到PID后终止
# taskkill /F /PID <PID>

# 前端端口：8080
netstat -ano | findstr :8080
# taskkill /F /PID <PID>
```

### 重启服务
```bash
# 先停止所有Node.js进程
taskkill /F /IM node.exe

# 然后分别启动前后端
# 启动后端
cd backend && npm start

# 启动前端
cd frontend && npm run serve
```
